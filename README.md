# 🔢 Contador

Projeto desenvolvido com **HTML5, CSS3 e JavaScript** que realiza contagem numérica de forma dinâmica e interativa.

## 🚀 Funcionalidades

- Contagem automática
- Incremento numérico
- Interface simples e responsiva
- Validação de valores

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
